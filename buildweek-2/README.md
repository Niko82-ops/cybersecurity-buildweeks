# Cyber Security Training Projects

This repository contains the projects developed during the Cyber Security training program.

The projects are organized by build weeks and include practical activities related to networking, penetration testing, and security analysis.

## Build Week 1 – Networking

Projects related to network analysis and scanning techniques.

Topics covered:
- HTTP scanning
- Port scanning
- Socket programming
- Network monitoring

## Build Week 2 – Penetration Testing

Practical penetration testing activities and vulnerability exploitation.

Topics covered:
- Penetration testing
- Black-box testing
- Windows exploitation
- Buffer overflow
- XSS attacks

## Structure

Each build week contains folders with the related reports for each task.
