
# Build Week 1 – Networking Projects

This folder contains the networking projects developed during the first build week of the Cyber Security training program.

## Projects

• HTTP Scanner – Analysis and development of a basic HTTP scanning tool.  
• Port Scanner – Implementation of a port scanning script and analysis of open ports.  
• Socket Sniffer – Traffic monitoring and protocol analysis using Python sockets.  
• Theta Project – Network security proposal for an enterprise environment.

Each folder contains the report related to the activity.
